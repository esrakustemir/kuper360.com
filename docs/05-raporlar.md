# Kuper360 Raporlar ve Karar Ekranı

## Amaç

Kuper360 içindeki ürün, CRM, pazaryeri ve ithalat/ihracat verilerini kullanarak firmanın ticari durumunu görünür hale getirmek.

Bu modül muhasebe yapmaz. Nebim'deki resmi muhasebenin yerine geçmez.

Amaç yönetime karar verdirecek ticari raporlar oluşturmaktır.

## Ana Rapor Grupları

### Ürün Raporları

* Toplam ürün sayısı
* Aktif ürün sayısı
* Pasif ürün sayısı
* Eksik bilgili ürünler
* Görselsiz ürünler
* HS Code eksik ürünler
* Koli bilgisi eksik ürünler

### Karlılık Raporları

* Ürün bazlı kar
* Pazaryeri bazlı kar
* Platform bazlı kar karşılaştırması
* En karlı ürünler
* En düşük marjlı ürünler
* Zarar eden ürünler
* Başabaş fiyatın altında kalan ürünler

### CRM Raporları

* Toplam firma sayısı
* Müşteri sayısı
* Tedarikçi sayısı
* Ülke bazlı müşteri dağılımı
* Son görüşme tarihi geçmiş firmalar
* Potansiyel müşteri listesi

### İthalat / İhracat Raporları

* Açık proformalar
* Tamamlanan proformalar
* Bekleyen sevkiyatlar
* Ülke bazlı ihracat potansiyeli
* Tedarikçi bazlı ithalat takibi
* Belge durumu eksik işlemler

## Dashboard Kartları

Ana ekranda şu kartlar olmalıdır:

* Toplam ürün
* Aktif müşteri
* Devam eden işlem
* Bu ay tahmini ciro
* Zarar eden ürün sayısı
* Eksik bilgili ürün sayısı
* Bekleyen proforma sayısı
* Kritik takip gerektiren işlem sayısı

## Karar Ekranı

Yönetici sisteme girdiğinde şunları hızlıca görebilmelidir:

* Hangi ürünler para kazandırıyor?
* Hangi ürünler zarar ettiriyor?
* Hangi ürünlerin bilgisi eksik?
* Hangi müşteriler takip bekliyor?
* Hangi ihracat/ithalat işlemleri açık?
* Hangi pazaryerinde hangi ürün daha karlı?

## Filtreler

Raporlarda şu filtreler olmalıdır:

* Tarih aralığı
* Marka
* Kategori
* Ürün
* Müşteri
* Tedarikçi
* Ülke
* Pazaryeri
* Para birimi

## Çıktılar

* Ekran raporu
* Excel dışa aktarma
* PDF dışa aktarma
* Yönetici özet ekranı
